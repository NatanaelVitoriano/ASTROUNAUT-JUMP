# Astronaut Jump Game

## Descrição
Astronaut Jump é um jogo divertido inspirado no estilo de "Sky Jump". O objetivo do jogo é controlar um astronauta que salta entre plataformas, coletando pontos e evitando cair. O jogo é acessível através de um link que pode ser compartilhado via QR code, tornando-o perfeito para eventos como aniversários.

## Estrutura do Projeto
O projeto contém os seguintes arquivos:

- **`js/main.js`**: Contém a lógica do jogo, incluindo movimento do jogador, detecção de colisões e renderização dos elementos do jogo no canvas.
- **`css/style.css`**: Define os estilos do jogo, incluindo o fundo, layout e aparência visual dos elementos do jogo.
- **`index.html`**: Documento HTML principal que serve como ponto de entrada para o jogo. Inclui o elemento canvas para renderização do jogo e links para os arquivos JavaScript e CSS.

## Como Executar o Jogo
1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em um navegador da web.
3. Use as setas do teclado para mover o astronauta para a esquerda e para a direita. O jogador pode pular nas plataformas para ganhar pontos.

## Recursos
- Jogo responsivo que se adapta a diferentes tamanhos de tela.
- Pontuação baseada na altura alcançada pelo jogador.
- Colisões realistas com plataformas.

## Implantação no Vercel
Para implantar o projeto no Vercel:
1. Crie uma conta no Vercel.
2. Faça o upload do projeto.
3. O Vercel detectará automaticamente a estrutura do projeto e implantará o site.
4. Utilize um serviço de geração de QR code para criar um código que redirecione para a URL implantada do jogo.
